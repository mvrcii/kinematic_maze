Fixed Datasets:

LiebersLabStudy21

The LiebersLabStudy21 dataset focuses on VR users per- forming specific bowling and archery motions.

Dataset
Paper
Coordinate Syst.: RUB
Rotation Repr.: Euler (deg;XYZ;intr.)
Units: Meter
Time: rel. (ms)
File Format: CSV

LiebersBeatSaber23

The LiebersBeatSaber23 dataset of 15 users is based on the game Beatsaber and is aimed at user identification through motion.

Dataset
Paper
Coordinate Syst.: RUB
Rotation Repr.: Euler (deg;XYZ;intr.)/Quat.
Units: Meter
Time: 90 Hz (presumably fixed)
File Format: CSV

LiebersHand22

The ‘LiebersHand22’ dataset [2] includes interactions of 16 users with various interface elements in AR and VR environments for motion-based user identification

Dataset
Paper
Coordinate Syst.: RUB
Rotation Repr.: Euler (deg;XYZ;intr.)/Quat.
Units: Meter
Time: rel. (s and ms)
File Format: TSV

MooreCrossDomain23

MooreCrossDomain23 includes motion data of 45 VR users performing assembly tasks, also intended for user identification re- search

Dataset
Paper
Coordinate Syst.: RUB
Rotation Repr.: Euler (deg;XYZ;intr.)/Quat./Rot. Mat.
Units: Meter
Time: el. (s)
File Format: CSV

Vr.net

VR.net was designed for cybersickness research and provides 16 users playing various VR games.

Dataset
Paper 1
Coordinate Syst.: RUF
Rotation Repr.: Trans. Mat.
Units: Meter
Time: abs. (unix)
File Format: CSV

RMillerBall22

RMillerBall22 captures the motions of 41 users performing ball throwing actions in VR, segregated by user, device, session, and peripheral.

Dataset
Paper 1
Paper 2

Rotation Repr.: Qiaternion
Units: Meter
